## 🆚 Difference Between FOD and Dokumentasi Lapangan

| Feature | FOD | Dokumentasi Lapangan |
|---------|-----|----------------------|
| **Purpose** | Specific item findings | General activity docs |
| **Images Limit** | Unlimited | 2 max |
| **Item Linking** | Yes (dropdown) | No |
| **UI Complexity** | Complex | Simple |
| **PDF Location** | After Dokumentasi | Before FOD |
| **Use Case** | Link to specific defects | Document overall activity |
